# Story 38.3.3: PDF Guardian-Native Labels + ISO Disclaimer

## Description

Add Guardian-native dimension label treatment and ISO disclaimer footer to the PDF report. Guardian-native dimensions (Clinical Risk, Vendor Capability, Ethical Considerations, Sustainability) have no ISO mapping and should display a visual indicator "Guardian Healthcare-Specific" in the dimension table. The footer gets an ISO disclaimer per PRD Section 13.

## Acceptance Criteria

- [ ] Guardian-native dimensions show a small "Guardian Healthcare-Specific" label beneath the dimension name
- [ ] Non-Guardian-native dimensions show dimension name only (no label change)
- [ ] Footer includes ISO disclaimer text from `isoMessagingTerms.ts`
- [ ] Disclaimer text passes prohibited terms check (no "compliant"/"certified")
- [ ] Visual treatment is subtle (not distracting from scores)
- [ ] Under 300 LOC for ScoringPDFExporter.ts

### 2. Update ScoringPDFExporter.renderTemplate()

**File:** `packages/backend/src/infrastructure/export/ScoringPDFExporter.ts` (MODIFY)

Import the disclaimer constant and inject into template:

```typescript
import { ISO_DISCLAIMER } from '../../domain/compliance/isoMessagingTerms.js';

// In renderTemplate():

// Add Guardian-native label to dimension names
const dimensionScoresWithLabels = payload.dimensionScores.map((d) => {
  const isoData = data.dimensionISOData.find((iso) => iso.dimension === d.dimension);
  const baseLabel = DIMENSION_CONFIG[d.dimension as keyof typeof DIMENSION_CONFIG]?.label || d.dimension;

  // Add Guardian-native indicator beneath dimension name
  const labelHtml = isoData?.isGuardianNative
    ? `${this.escapeHtml(baseLabel)}<span class="guardian-native-label">Guardian Healthcare-Specific</span>`
    : this.escapeHtml(baseLabel);

  return {
    ...d,
    label: labelHtml, // Changed from text to HTML
    // ... confidence and ISO ref data from 38.3.1 ...
  };
});

// Inject ISO disclaimer
html = html.replace(/{{isoDisclaimer}}/g, this.escapeHtml(ISO_DISCLAIMER));
```

### 3. Key Rules

- **Template variable for label**: Since labels now contain HTML (the `<span>` for Guardian-native), use `{{{label}}}` (triple braces for raw HTML) instead of `{{label}}` in the dimension table rows.
- **ISO_DISCLAIMER import**: Use the constant from `isoMessagingTerms.ts` (Story 38.2.4). This ensures single source of truth for disclaimer text.
- **Escape disclaimer text**: Even though the disclaimer is a constant, use `escapeHtml()` for defense-in-depth.
- **Subtle visual treatment**: Guardian-native label is small (10px), italic, and purple (brand color). It should not dominate the dimension name.
