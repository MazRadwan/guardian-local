# Story 38.4.3: Word Guardian-Native Labels + ISO Disclaimer

## Description

Add Guardian-native dimension label treatment and ISO disclaimer footer to the Word document. Guardian-native dimensions get a small italic label "Guardian Healthcare-Specific" beneath the dimension name in the table. The footer gets an ISO disclaimer. Mirrors PDF Story 38.3.3.

## Acceptance Criteria

- [ ] Guardian-native dimensions show "Guardian Healthcare-Specific" sublabel in dimension name cell
- [ ] Non-Guardian-native dimensions show dimension name only
- [ ] Footer includes ISO disclaimer text from `isoMessagingTerms.ts`
- [ ] Sublabel is smaller font, italic, purple (brand color)
- [ ] Under 300 LOC per file (or split if needed)

## Technical Approach

### 1. Update createDimensionTable in WordSectionBuilders.ts

**File:** `packages/backend/src/infrastructure/export/WordSectionBuilders.ts` (MODIFY)

Update the dimension name cell to include Guardian-native sublabel:

```typescript
// In createDimensionTable, update the dimension name cell:
const isoData = data.dimensionISOData.find((iso) => iso.dimension === d.dimension);
const dimLabel = DIMENSION_CONFIG[d.dimension as keyof typeof DIMENSION_CONFIG]?.label || d.dimension;

const dimCellChildren: Paragraph[] = [
  new Paragraph({
    children: [new TextRun({ text: dimLabel, size: 22 })],
  }),
];

if (isoData?.isGuardianNative) {
  dimCellChildren.push(
    new Paragraph({
      children: [new TextRun({
        text: 'Guardian Healthcare-Specific',
        size: 16,
        italics: true,
        color: BRAND_COLOR,
      })],
      spacing: { before: 40 },
    })
  );
}

new TableCell({
  children: dimCellChildren,
  shading: { type: ShadingType.CLEAR, fill: isEven ? 'FFFFFF' : 'F9FAFB' },
});
```

### 2. Update ScoringWordExporter.ts footer

**File:** `packages/backend/src/infrastructure/export/ScoringWordExporter.ts` (MODIFY)

Import and add ISO disclaimer to the document footer:

```typescript
import { ISO_DISCLAIMER } from '../../domain/compliance/isoMessagingTerms.js';

// In generateWord, update the footer:
footers: {
  default: new Footer({
    children: [
      new Paragraph({
        alignment: AlignmentType.CENTER,
        children: [
          new TextRun({
            text: `Generated by Guardian | Rubric: ${data.report.rubricVersion} | `,
            size: 18,
            color: '9CA3AF',
          }),
          // ... existing page number runs ...
        ],
      }),
      // NEW: ISO disclaimer
      new Paragraph({
        alignment: AlignmentType.CENTER,
        children: [
          new TextRun({
            text: ISO_DISCLAIMER,
            size: 16,
            color: '9CA3AF',
            italics: true,
          }),
        ],
        spacing: { before: 100 },
      }),
    ],
  }),
},
```

### 3. Key Rules

- **Same visual language as PDF**: Purple italic text for Guardian-native label. Matches PDF treatment from Story 38.3.3.
- **ISO_DISCLAIMER constant**: Import from `isoMessagingTerms.ts` (Story 38.2.4). Single source of truth.
- **LOC management**: If `WordSectionBuilders.ts` is approaching 300 LOC, consider extracting Guardian-native logic to separate function.
