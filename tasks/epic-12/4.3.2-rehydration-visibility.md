## Implementation

### 1. useQuestionnaireCardVisibility.ts

```typescript
import { RefObject, useState, useEffect } from 'react';

/**
 * Tracks whether the questionnaire card is visible in the message list viewport.
 * Uses IntersectionObserver with the container as the root.
 *
 * @param cardRef - Ref to the QuestionnairePromptCard element
 * @param containerRef - Ref to the scrollable message list container
 * @returns boolean - true if card is visible (or refs not ready), false if scrolled out
 */
export function useQuestionnaireCardVisibility(
  cardRef: RefObject<HTMLDivElement | null>,
  containerRef: RefObject<HTMLDivElement | null>
): boolean {
  const [isVisible, setIsVisible] = useState(true); // Default visible until proven otherwise

  useEffect(() => {
    const card = cardRef.current;
    const container = containerRef.current;

    // If refs aren't ready, assume visible
    if (!card || !container) return;

    const observer = new IntersectionObserver(
      ([entry]) => setIsVisible(entry.isIntersecting),
      {
        root: container,
        threshold: 0.1, // Consider visible if 10% is showing
      }
    );

    observer.observe(card);
    return () => observer.disconnect();
  }, [cardRef, containerRef]);

  return isVisible;
}
```

**IMPORTANT:** Hook must be called at component top level, NOT inside an effect.


Add rehydration effect:
```typescript
// Rehydrate questionnaire state from localStorage on conversation switch
useEffect(() => {
  if (!activeConversationId || !user?.id) return;

  // Always clear previous conversation's state first
  useChatStore.getState().clearPendingQuestionnaire();
  useChatStore.getState().setQuestionnaireUIState('hidden');

  // If dismissed for this conversation, don't restore
  if (persistence.isDismissed(activeConversationId)) {
    return;
  }

  // Try to load persisted payload
  const savedPayload = persistence.loadPayload(activeConversationId);
  if (savedPayload) {
    useChatStore.getState().setPendingQuestionnaire(savedPayload);
    useChatStore.getState().setQuestionnaireUIState('ready');
  }
}, [activeConversationId, user?.id, persistence]);
```

## Key Implementation Notes

### Hook Placement
- `useQuestionnairePersistence` MUST be called at component top level
- NOT inside useEffect or conditionally
- The persistence object is stable (same functions each render for same userId)

### Conversation Switch Behavior
1. Clear previous conversation's pendingQuestionnaire
2. Clear previous conversation's questionnaireUIState → 'hidden'
3. Check if current conversation was dismissed
4. If not dismissed, restore from localStorage

### SSR Safety
- `useQuestionnairePersistence` already handles SSR with `typeof window !== 'undefined'`
- Effect only runs client-side (useEffect doesn't run during SSR)

### Ref Threading (for Story 4.3.3+)
- `ChatInterface` will create `questionnaireCardRef = useRef<HTMLDivElement>(null)`
- `messageListRef` should already exist for scroll handling
- Both refs passed to `useQuestionnaireCardVisibility(questionnaireCardRef, messageListRef)`
