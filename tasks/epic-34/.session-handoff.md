## What This Epic Does

Extracts `executeConsultToolLoop()` and `buildAugmentedMessages()` from MessageHandler.ts into a dedicated `ConsultToolLoopService`.

**That's it.** Do not refactor anything else.

## Why This Epic Exists

1. MessageHandler.ts is 1656 LOC (violates 300 LOC limit)
2. Tool loop is ~350 LOC of isolated logic
3. Tool loop is not working great and needs iteration
4. Extracting it doesn't touch sensitive optimized code
