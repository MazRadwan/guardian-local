# Story 13.5.4 - Remove Simulated Progression

## Status: Complete

## Context
Backend now emits real phase events (13.5.2) and frontend listens (13.5.3). This story removes the temporary timer-based simulation from 13.4.5.

## Goal
Remove all simulation code from ChatInterface, keeping only the initial step setting. Backend events now drive progression.

## File to Modify
`apps/web/src/components/chat/ChatInterface.tsx`

## Implementation

### Code to Remove
Find and remove all code marked with:
- `// TEMPORARY`
- `// TODO: Remove in Story 13.5`

Specifically remove:
1. `simulationRef` declaration
2. `setInterval` simulation logic in `handleGenerateQuestionnaire`
3. Timer cleanup effect for unmount
4. Timer cleanup effect for conversation switch
5. Conversation gating logic inside interval

### Before (13.4.5 simulation)
```typescript
const simulationRef = useRef<NodeJS.Timeout | null>(null);

const handleGenerateQuestionnaire = useCallback(() => {
  const conversationIdAtStart = activeConversationId;

  resetGenerationStep();
  setCurrentGenerationStep(0);
  setQuestionnaireUIState('generating');

  // TEMPORARY: Simulate step progression
  let step = 0;
  simulationRef.current = setInterval(() => {
    const currentConversationId = useChatStore.getState().activeConversationId;
    if (currentConversationId !== conversationIdAtStart) {
      clearInterval(simulationRef.current!);
      simulationRef.current = null;
      return;
    }
    step++;
    if (step < GENERATION_STEPS.length) {
      setCurrentGenerationStep(step);
    } else {
      clearInterval(simulationRef.current!);
      simulationRef.current = null;
    }
  }, 1500);

  socket?.emit('generate_questionnaire', { ... });
}, [/* deps */]);

// Cleanup on unmount
useEffect(() => {
  return () => {
    if (simulationRef.current) {
      clearInterval(simulationRef.current);
      simulationRef.current = null;
    }
  };
}, []);

// Cleanup on conversation switch
useEffect(() => {
  if (simulationRef.current) {
    clearInterval(simulationRef.current);
    simulationRef.current = null;
  }
}, [activeConversationId]);
```

### After (backend-driven)
```typescript
const handleGenerateQuestionnaire = useCallback(() => {
  // Start at step 0 immediately (backend will emit phase 0 shortly)
  resetGenerationStep();
  setCurrentGenerationStep(0);
  setQuestionnaireUIState('generating');

  // Emit generation request - backend handles phase progression
  socket?.emit('generate_questionnaire', {
    conversationId: activeConversationId,
    assessmentType: pendingQuestionnaire?.assessmentType,
    vendorName: pendingQuestionnaire?.vendorName,
    productDescription: pendingQuestionnaire?.productDescription,
    questionCount: pendingQuestionnaire?.questionCount,
  });
}, [
  socket,
  activeConversationId,
  pendingQuestionnaire,
  resetGenerationStep,
  setCurrentGenerationStep,
  setQuestionnaireUIState,
]);

// No simulation refs or cleanup effects needed
```

### What to Keep
- `resetGenerationStep()` - clears previous state
- `setCurrentGenerationStep(0)` - shows immediate feedback
- `setQuestionnaireUIState('generating')` - UI state change
- `socket?.emit('generate_questionnaire', {...})` - backend request
