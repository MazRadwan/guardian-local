# Story 13.4.5 - Simulated Step Progression

## Status: Not Started

## Context
The stepper UI is integrated (13.4.3, 13.4.4) but steps don't progress yet. This story adds temporary timer-based simulation to demonstrate the UI. Will be replaced by real backend events in Story 13.5.

## Goal
Temporarily simulate step progression using timers when generation starts, completing when `export_ready` event arrives.

## Files to Modify
- `apps/web/src/components/chat/ChatInterface.tsx`

## Implementation

### ChatInterface.tsx - Simulate Steps on Generate

```typescript
import { useChatStore, GENERATION_STEPS } from '@/stores/chatStore';

// Get actions
const { setCurrentGenerationStep, resetGenerationStep } = useChatStore(
  (state) => ({
    setCurrentGenerationStep: state.setCurrentGenerationStep,
    resetGenerationStep: state.resetGenerationStep,
  })
);

// Track simulation interval for cleanup
const simulationRef = useRef<NodeJS.Timeout | null>(null);

// In handleGenerateQuestionnaire:
const handleGenerateQuestionnaire = useCallback(() => {
  const conversationIdAtStart = activeConversationId;  // Capture for gating

  // Reset and start at step 0
  resetGenerationStep();
  setCurrentGenerationStep(0);
  setQuestionnaireUIState('generating');

  // TEMPORARY: Simulate step progression
  // TODO: Remove in Story 13.5 when backend emits real phase events
  // See: tasks/epic-13/13.5-stepper-backend-orchestration.md
  // Gated to active conversation to prevent cross-convo state updates
  let step = 0;
  simulationRef.current = setInterval(() => {
    // Guard: Stop if conversation changed
    const currentConversationId = useChatStore.getState().activeConversationId;
    if (currentConversationId !== conversationIdAtStart) {
      clearInterval(simulationRef.current!);
      simulationRef.current = null;
      return;
    }

    step++;
    if (step < GENERATION_STEPS.length) {
      setCurrentGenerationStep(step);
    } else {
      // Stop at last step - export_ready will mark complete
      clearInterval(simulationRef.current!);
      simulationRef.current = null;
      // Don't set complete here - let export_ready event do it
    }
  }, 1500); // 1.5 seconds per step

  // Emit actual generation request
  socket?.emit('generate_questionnaire', {
    conversationId: activeConversationId,
    assessmentType: pendingQuestionnaire?.assessmentType,
    // ... other params
  });
}, [/* deps */]);

// Cleanup on unmount
useEffect(() => {
  return () => {
    if (simulationRef.current) {
      clearInterval(simulationRef.current);
      simulationRef.current = null;
    }
  };
}, []);

// Cleanup on conversation switch (in addition to guard)
useEffect(() => {
  if (simulationRef.current) {
    clearInterval(simulationRef.current);
    simulationRef.current = null;
  }
}, [activeConversationId]);

// Cleanup on error
const handleQuestionnaireError = useCallback((error: string) => {
  if (simulationRef.current) {
    clearInterval(simulationRef.current);
    simulationRef.current = null;
  }
  setQuestionnaireUIState('error');
  setQuestionnaireError(error);
}, []);
```

### Reset on Error
```typescript
// In handleExtractionFailed or error handler:
const handleExtractionFailed = useCallback((data: { conversationId: string; error: string }) => {
  if (data.conversationId !== activeConversationId) return;

  resetGenerationStep();  // Reset to -1
  setQuestionnaireUIState('error');
  setQuestionnaireError(data.error);
}, [activeConversationId]);
```

## Timing

| Step | Time | Label |
|------|------|-------|
| 0 | 0s | Context gathered |
| 1 | 1.5s | Generating questions |
| 2 | 3s | Validating structure |
| 3 | 4.5s | Saving assessment |
| Complete | export_ready event | N/A |

Note: Actual generation typically takes 10-30 seconds. The simulation will reach step 3 and wait for the real `export_ready` event.

## Robustness Guards

| Guard | Trigger | Action |
|-------|---------|--------|
| Conversation change | activeConversationId differs | Stop timer, clear ref |
| Unmount | Component unmounts | Stop timer, clear ref |
| Error | extraction_failed event | Stop timer, reset step to -1 |
| Completion | export_ready event | Set step to length (complete) |
