## Requirements
- On conversation switch or page reload, rehydrate questionnaire UI state per conversation:
  - If `exportReadyByConversation[conversationId]` exists → restore 'download' state
  - Else if localStorage payload exists → restore 'ready' state
  - Else → 'hidden' state
- **Ignore** any legacy dismiss/hide flag; do not check or apply dismissals
- Priority: export_ready > questionnaire_ready > nothing
- Preserve existing generation flow semantics
