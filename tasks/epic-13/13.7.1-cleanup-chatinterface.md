# Story 13.7.1 - Clean Up ChatInterface.tsx

## Status: Not Started

## Context
The sticky indicator is no longer needed since the inline QuestionnairePromptCard is now the sole status surface. This story removes all sticky indicator references from ChatInterface.

## Goal
Remove all sticky indicator imports, refs, hook calls, handlers, and render blocks from ChatInterface.tsx.

## File to Modify
`apps/web/src/components/chat/ChatInterface.tsx`

