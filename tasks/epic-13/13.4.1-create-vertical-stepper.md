# Story 13.4.1 - Create VerticalStepper Component

## Status: Not Started

## Context
Story 13.4 adds a progress stepper to the questionnaire generation UI. This sub-story creates the reusable VerticalStepper component based on the Appendix demo code.

## Goal
Create a standalone VerticalStepper component that renders animated steps with checkmarks, pulse indicators, and connecting lines.

## File to Create
`apps/web/src/components/chat/VerticalStepper.tsx` (NEW)

## Implementation

### Types
```typescript
export interface Step {
  id: string;
  label: string;
}

interface VerticalStepperProps {
  steps: Step[];
  currentStep: number;  // -1 = none, 0-N = active step index
  isRunning: boolean;
}
```

### Component Structure
```typescript
import React from 'react';

export const VerticalStepper: React.FC<VerticalStepperProps> = ({
  steps,
  currentStep,
  isRunning,
}) => (
  <div className="space-y-0 py-1">
    {steps.slice(0, Math.max(currentStep + 1, 0)).map((step, idx) => {
      const isComplete = idx < currentStep;
      const isActive = idx === currentStep && isRunning;
      const isLast = idx === Math.min(currentStep, steps.length - 1);

      return (
        <div
          key={step.id}
          className="flex items-stretch animate-fadeIn"
          style={{ animationDelay: `${idx * 100}ms` }}
        >
          {/* Left indicator column */}
          <div className="flex flex-col items-center mr-3">
            <div
              className={`
                w-5 h-5 rounded-full flex items-center justify-center text-xs transition-all duration-300
                ${isComplete ? 'bg-emerald-500 text-white' : ''}
                ${isActive ? 'bg-blue-500 text-white ring-2 ring-blue-200' : ''}
              `}
            >
              {isComplete ? (
                <svg className="w-3 h-3" fill="none" viewBox="0 0 24 24" stroke="currentColor" strokeWidth={3}>
                  <path strokeLinecap="round" strokeLinejoin="round" d="M5 13l4 4L19 7" />
                </svg>
              ) : isActive ? (
                <div className="w-2 h-2 bg-white rounded-full animate-pulse" />
              ) : null}
            </div>
            {/* Connecting line */}
            {!isLast && (
              <div className={`w-0.5 flex-1 min-h-[16px] transition-colors duration-300 ${isComplete ? 'bg-emerald-300' : 'bg-slate-200'}`} />
            )}
          </div>
          {/* Step label */}
          <div className={`pb-3 ${isLast ? 'pb-0' : ''}`}>
            <div className={`
              text-sm transition-colors duration-300
              ${isComplete ? 'text-slate-600' : ''}
              ${isActive ? 'text-slate-700 font-medium' : ''}
            `}>
              {step.label}
              {isActive && <span className="text-blue-500 animate-pulse">...</span>}
            </div>
          </div>
        </div>
      );
    })}
  </div>
);
```

### CSS Animation
Add to `apps/web/src/app/globals.css` or use inline style:
```css
@keyframes fadeIn {
  from { opacity: 0; transform: translateY(-4px); }
  to { opacity: 1; transform: translateY(0); }
}
.animate-fadeIn {
  animation: fadeIn 0.3s ease-out forwards;
}
```

Note: Tailwind's `animate-fadeIn` may need to be added to `tailwind.config.ts` if not available.

## Visual States

| State | Indicator | Line Color | Text Style |
|-------|-----------|------------|------------|

| Active | Blue pulse | Gray | Bold + "..." |
| Future | Not rendered | N/A | N/A |

```typescript
export const VerticalStepper = React.memo<VerticalStepperProps>(({ steps, currentStep, isRunning }) => {
  // ...
});
```

## Acceptance Criteria
- [ ] Component renders steps up to currentStep (inclusive)
- [ ] Completed steps show green checkmark icon
- [ ] Active step shows blue pulsing indicator
- [ ] Steps animate in with fadeIn + staggered delay (100ms per step)
- [ ] Connecting lines fill green for completed steps
- [ ] No external dependencies (standalone component)
- [ ] TypeScript types exported for reuse
- [ ] No rendering when currentStep is -1
- [ ] **Component wrapped in React.memo** for performance
