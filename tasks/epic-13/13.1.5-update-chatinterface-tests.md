# Story 13.1.6 - Update ChatInterface Tests

## Status: Not Started

## Context
Story 13.1.3 removed the `handleDismiss` function and `onDismiss` prop from ChatInterface. The tests need to be updated to reflect this change.

## Goal
Update ChatInterface tests to remove dismiss-related test cases while keeping rehydration tests intact.

## File to Modify
`apps/web/src/components/chat/__tests__/ChatInterface.test.tsx`

## Current Test Coverage
The existing tests (1442 lines) cover:
1. Questionnaire inline card rendering in slot
2. Conditional rendering (conversationId match, UI state)
3. Rehydration on conversation switch
4. **Dismissal behavior** (NEEDS UPDATES)
5. Persistence integration
6. Sticky indicator tests (disabled)

### 2. Update Mock Verification
Remove expectations that check for `onDismiss` prop:

**Before:**
```tsx

  expect.objectContaining({
    onGenerate: expect.any(Function),
```

**After:**

### 3. Keep Rehydration Tests

