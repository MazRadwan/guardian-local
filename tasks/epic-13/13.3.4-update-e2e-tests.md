# Story 13.3.4 - Update E2E Tests for Durable Downloads

## Status: Not Started

## Context
Stories 13.3.1-13.3.3 changed download behavior so cards remain visible after download. E2E tests may have assertions expecting the card to disappear.

## Goal
Update e2e tests to verify durable download behavior and add assertions for re-download capability.

## File to Modify
`apps/web/e2e/questionnaire-generation.spec.ts`

## Current Test Coverage
The e2e tests likely verify:
1. Questionnaire generation flow
2. Download button clicks
3. **Card disappearance after download** (NEEDS UPDATE)

## Required Changes

### 1. Find and Update Card Disappearance Assertions

Search for assertions like:

```typescript
// BEFORE: Expecting card to disappear
await page.locator('[data-testid="download-pdf-btn"]').click();
await expect(page.locator('[data-testid="questionnaire-card-download"]')).not.toBeVisible();
```

Update to:

```typescript
// AFTER: Card stays visible
await page.locator('[data-testid="download-pdf-btn"]').click();
// Wait for download to complete (check for success indicator or timeout)
await page.waitForTimeout(1000); // or wait for specific success state
// Card should STILL be visible
await expect(page.locator('[data-testid="questionnaire-card-download"]')).toBeVisible();
```

### 2. Add Multi-Download Test

```typescript
test('should allow multiple downloads without regenerating', async ({ page }) => {
  // Setup: Navigate to conversation with completed questionnaire
  await page.goto('/chat/conv-123');

  // Wait for download card to appear
  await expect(page.locator('[data-testid="questionnaire-card-download"]')).toBeVisible();

  // Download PDF
  const downloadPromise1 = page.waitForEvent('download');
  await page.locator('[data-testid="download-pdf-btn"]').click();
  const download1 = await downloadPromise1;
  expect(download1.suggestedFilename()).toContain('.pdf');

  // Verify card still visible
  await expect(page.locator('[data-testid="questionnaire-card-download"]')).toBeVisible();
  await expect(page.locator('[data-testid="download-word-btn"]')).toBeVisible();

  // Download Word
  const downloadPromise2 = page.waitForEvent('download');
  await page.locator('[data-testid="download-word-btn"]').click();
  const download2 = await downloadPromise2;
  expect(download2.suggestedFilename()).toContain('.docx');

  // Verify card STILL visible
  await expect(page.locator('[data-testid="questionnaire-card-download"]')).toBeVisible();
});
```

### 3. Add Page Reload Persistence Test

```typescript
test('should restore download state after page reload', async ({ page }) => {
  // Setup: Navigate to conversation with completed questionnaire
  await page.goto('/chat/conv-123');

  // Wait for download card
  await expect(page.locator('[data-testid="questionnaire-card-download"]')).toBeVisible();

  // Download to confirm export state is saved
  const downloadPromise = page.waitForEvent('download');
  await page.locator('[data-testid="download-pdf-btn"]').click();
  await downloadPromise;

  // Reload page
  await page.reload();

  // Verify download card restored
  await expect(page.locator('[data-testid="questionnaire-card-download"]')).toBeVisible();
  await expect(page.locator('[data-testid="download-pdf-btn"]')).toBeVisible();
});
```

## Common Patterns to Update

| Old Assertion | New Assertion |
|---------------|---------------|
| `expect(card).not.toBeVisible()` after download | `expect(card).toBeVisible()` after download |
| `expect(card).toBeHidden()` after download | `expect(card).toBeVisible()` after download |
| Waiting for card to disappear | Waiting for download to complete |
