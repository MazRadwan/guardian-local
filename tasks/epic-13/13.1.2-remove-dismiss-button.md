# Story 13.1.2 - Remove Dismiss Button and Handler

## Status: Not Started

## Context
The `QuestionnairePromptCard` currently has a dismiss (X) button that allows users to hide the card. Since the card is now an inline chat bubble (not a sticky footer), users can simply scroll past it - explicit dismissal is unnecessary and adds visual clutter.

## Goal
Remove the dismiss button and `onDismiss` prop from the QuestionnairePromptCard component.

## File to Modify
`apps/web/src/components/chat/QuestionnairePromptCard.tsx`

## Current Implementation
The component currently has:
```tsx
interface QuestionnairePromptCardProps {
  payload: QuestionnaireReadyPayload;
  uiState: QuestionnaireUIState;
  error?: string | null;
  exportData?: { formats: string[]; assessmentId: string } | null;
  onGenerate: () => void;
  onDismiss: () => void;  // <-- Remove this
  onDownload: (format: string) => void;
  onRetry: () => void;
  className?: string;
}
```

And in the JSX, there's a dismiss button (usually an X icon button in the header area).

## Required Changes

### 1. Remove onDismiss from Props Interface
```tsx
interface QuestionnairePromptCardProps {
  payload: QuestionnaireReadyPayload;
  uiState: QuestionnaireUIState;
  error?: string | null;
  exportData?: { formats: string[]; assessmentId: string } | null;
  onGenerate: () => void;
  // onDismiss: () => void;  <-- REMOVED
  onDownload: (format: string) => void;
  onRetry: () => void;
  className?: string;
}
```

### 3. Remove Dismiss Button JSX
Find and remove the dismiss button. It's typically an icon button with:
- `X` icon from lucide-react
- `onClick={onDismiss}` handler
- Located in the card header area

Example of what to remove:
```tsx
{/* Remove this entire button */}
<button
  onClick={onDismiss}
  className="..."
  aria-label="Dismiss"
>
  <X className="h-4 w-4" />
</button>
```

### 4. Remove X Icon Import (if unused elsewhere)
If `X` from lucide-react is only used for the dismiss button, remove it from imports:
```tsx
import { FileText, Loader2, Download, AlertCircle, RefreshCw } from 'lucide-react';
// X is no longer imported
```
