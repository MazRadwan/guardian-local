# Story 13.4.4 - Add Collapse/Expand Behavior

## Status: Not Started

## Context
Story 13.4.3 integrated the stepper into QuestionnairePromptCard. This story adds collapsible panel behavior: auto-expand on start, auto-collapse on complete, manual toggle.

## Goal
Make the stepper section collapsible with a clickable header, auto-expand/collapse behavior, and summary when collapsed.

## File to Modify
`apps/web/src/components/chat/QuestionnairePromptCard.tsx`

## Implementation

### Add Local State
```typescript
const [isStepperExpanded, setIsStepperExpanded] = useState(true);
const isComplete = currentGenerationStep >= generationSteps.length;
```

### Auto-Expand on Generation Start
```typescript
useEffect(() => {
  if (isGenerating && currentGenerationStep >= 0) {
    setIsStepperExpanded(true);
  }
}, [isGenerating, currentGenerationStep]);
```

### Auto-Collapse on Completion
```typescript
useEffect(() => {
  if (isComplete) {
    const timer = setTimeout(() => setIsStepperExpanded(false), 800);
    return () => clearTimeout(timer);
  }
}, [isComplete]);
```

### Collapsible UI Structure
Replace the stepper section from 13.4.3 with:

```typescript
{(isGenerating || isComplete) && (
  <div className="mt-3 border border-slate-200 rounded-lg bg-white overflow-hidden">
    {/* Clickable header - always visible */}
    <button
      onClick={() => setIsStepperExpanded(!isStepperExpanded)}
      className="w-full flex items-center justify-between px-3 py-2.5 hover:bg-slate-50 transition-colors"
    >
      <div className="flex items-center gap-2">
        {isComplete ? (
          <>
            <div className="w-5 h-5 rounded-full bg-emerald-500 flex items-center justify-center">
              <svg className="w-3 h-3 text-white" fill="none" viewBox="0 0 24 24" stroke="currentColor" strokeWidth={3}>
                <path strokeLinecap="round" strokeLinejoin="round" d="M5 13l4 4L19 7" />
              </svg>
            </div>
            <span className="text-sm font-medium text-slate-700">Assessment Complete</span>
          </>
        ) : (
          <>
            <div className="w-5 h-5 border-2 border-blue-500 border-t-transparent rounded-full animate-spin" />
            <span className="text-sm font-medium text-slate-700">Generating Assessment</span>
          </>
        )}
      </div>
      {/* Chevron */}
      <svg
        className={`w-4 h-4 text-slate-400 transition-transform duration-200 ${isStepperExpanded ? 'rotate-180' : ''}`}
        fill="none"
        viewBox="0 0 24 24"
        stroke="currentColor"
      >
        <path strokeLinecap="round" strokeLinejoin="round" strokeWidth={2} d="M19 9l-7 7-7-7" />
      </svg>
    </button>

    {/* Collapsible stepper content */}
    <div className={`overflow-hidden transition-all duration-300 ease-in-out ${isStepperExpanded ? 'max-h-64 opacity-100' : 'max-h-0 opacity-0'}`}>
      <div className="px-3 pb-3 border-t border-slate-100">
        <div className="pt-3">
          <VerticalStepper
            steps={generationSteps}
            currentStep={currentGenerationStep}
            isRunning={isGenerating}
          />
        </div>
      </div>
    </div>

    {/* Summary when collapsed + complete */}
    {isComplete && !isStepperExpanded && (
      <div className="px-3 pb-2.5 -mt-1">
        <span className="text-xs text-slate-500">
          {payload.questionCount || 40} questions · {payload.assessmentType || 'Assessment'}
        </span>
      </div>
    )}
  </div>
)}
```
