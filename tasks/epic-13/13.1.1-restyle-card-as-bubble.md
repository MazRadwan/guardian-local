# Story 13.1.1 - Restyle QuestionnairePromptCard as Assistant Bubble

## Status: Not Started

## Context
The `QuestionnairePromptCard` component currently uses Shadcn UI's `Card` and `CardContent` components with colored borders for different assessment types. This story restylesit to look like an assistant chat bubble, matching the `ChatMessage` component's assistant styling.

## Goal
Change card styling from Shadcn Card to assistant-bubble style that visually matches assistant messages in the chat.

## File to Modify
`apps/web/src/components/chat/QuestionnairePromptCard.tsx`

## Current Implementation
The component currently:
- Uses Shadcn `Card` and `CardContent` components
- Has colored borders based on assessment type (green/blue/purple)
- Contains 4 UI states: ready, generating, download, error
- Has header with icon, title, and dismiss button
- Shows assessment type badge, context summary, question count

## Required Changes

### 1. Replace Shadcn Card with Plain Div
Remove imports and usage of `Card`/`CardContent` from `@/components/ui/card`.

### 2. Apply Assistant Bubble Styling
Replace the outer Card wrapper with a div using these classes:
```tsx
<div className="bg-slate-50 rounded-2xl rounded-tl-sm p-4 max-w-md border border-slate-100">
```

Styling breakdown:
- `bg-slate-50` - Light gray background matching assistant messages
- `rounded-2xl rounded-tl-sm` - Rounded corners with small top-left (bubble shape)
- `p-4` - 16px padding all around
- `max-w-md` - Max width of 448px (consistent with chat bubbles)
- `border border-slate-100` - Subtle border

### 3. Simplify Internal Layout
Keep the internal structure but adjust spacing:
- Remove CardContent wrapper
- Adjust internal padding/margins as needed
- Keep the flex layout for icon + content

### 4. Keep State-Specific Styling
Maintain visual distinction for states but integrate with bubble style:
- **Ready**: Default slate-50 background
- **Generating**: Add opacity-75 or subtle animation
- **Download**: Add green accent (emerald-500 border-left or icon)
- **Error**: Add red accent (red-500 border-left or icon)

### 5. Keep Assessment Type Badge
Maintain the colored badge for assessment type (quick/comprehensive/category_focused).

## Code Structure (After)

```tsx
// Remove these imports:
// import { Card, CardContent } from '@/components/ui/card';

// Main wrapper becomes:
<div
  ref={ref}
  data-testid={`questionnaire-card-${uiState}`}
  className={cn(
    "bg-slate-50 rounded-2xl rounded-tl-sm p-4 max-w-md border border-slate-100",
    uiState === 'generating' && "opacity-75",
    className
  )}
>
  {/* Keep existing internal content structure */}
</div>
```
